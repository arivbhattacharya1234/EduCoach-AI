# EduCoach-AI
EduCoach AI: Real-time AI coaching for Indian teachers. Empower educators like Sunita with instant, personalized support to overcome classroom challenges and inspire student learning.
## Theme Alignment
Theme 1: Lack of on-the-go, need-based support to teachers

EduCoach AI directly addresses the absence of real-time instructional
support by providing instant AI-powered guidance during live classroom
situations.
## Problem Statement
Teachers across India receive training in modern pedagogy such as FLN
and activity-based learning, but lack real-time, classroom-specific
support when lessons derail or students disengage.
## The Story Behind It
Imagine Sunita, a passionate teacher in India, trained in innovative

## Features
- Instant chat-based support for classroom challenges
- Personalized teaching tips and activity suggestions
- Multilingual support for Indian classrooms
- Offline-capable design for low-connectivity regions
- ## Tech Stack
- Frontend: React Native
- Backend: Node.js with Express
- AI: OpenAI GPT
- Database: Firebase
## Project Status
This project is currently a working prototype developed for the hackathon.
